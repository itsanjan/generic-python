# Data Structures
Collection :- list, tuple, set, dict

- List   : []
  - name = ['Anjan', 'Gopi', 'Divya']
  - 1D
  - Allows duplicate value 
  - Maintains order 
- Tuple : ()
  - name = ('Anjan', 'Gopi', 'Divya')
  - 1D
  - Allows duplicate value
  - Maintains order     
- Set   : {name}
  - 1D
  - Doesnt allow duplicate values
  - its unordered
- Dict  : {name : phone}
  - 2D
